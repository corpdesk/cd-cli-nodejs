# Getting Started

This chapter guides you through the initial setup of CD-CLI.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/corpdesk/cd-cli-nodejs.git
   ```
2. Navigate to the project directory:
   ```bash
   cd cd-cli-nodejs
   ```
3. Install dependencies:
   ```bash
   pnpm install
   ```
